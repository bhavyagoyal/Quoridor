Quoridor Bot
======
An adaptive AI player for finite time Quoridor N-M-K tournament. Bot has been developed as an instance of **minimax** search with alpha beta pruning using evaluation functions

## Requirements
* pythonTk package

## Usage
* Compile Bot
```
$ ./compile.sh
```
* Run Server
```
$ python TkBoard.py
```

* Run Bot1
```
$ ./run1.sh 127.0.0.1 12345
```

* Run Bot2:
```
$ ./run1.sh 127.0.0.1 12345
```
